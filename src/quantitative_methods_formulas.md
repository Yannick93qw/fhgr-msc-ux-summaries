# Formeln

## Relative Häufigkeit
\\[
\\%k = \frac{fk}{n}*100\\%
\\]

## Gültige Prozentwerte bei Häufigkeiten
fk = Absolute Häufigkeit

nv = Gültige Anzahl an Werten
\\[
\\%kv = \frac{fk}{nv}*100\\%
\\]


## Winkel eines Kreisauschnittes berechnen
\\[
\\gamma k = \frac{\\%k * 360\circ}{100\\%}
\\]
